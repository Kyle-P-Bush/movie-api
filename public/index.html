Movie API Documentation

Overview

The Movie API provides a simple RESTful interface for managing a list of movies. You can retrieve, create, update, and delete movies using standard HTTP methods (GET, POST, PUT, DELETE).

API Endpoints

GET /api/movies

Returns a list of all movies in the database.

GET /api/movies/:id

Returns the details of a specific movie by its ID.

POST /api/movies

Creates a new movie with the given title, year, and genre.

PUT /api/movies/:id

Updates the details of an existing movie by its ID.

DELETE /api/movies/:id

Deletes a movie by its ID.

Usage Examples

Get a list of movies

GET /api/movies

Get a specific movie by ID

GET /api/movies/1

Create a new movie

POST /api/movies
{
  "title": "Inception",
  "year": 2010,
  "genre": "Action, Adventure, Sci-Fi"
}

Update an existing movie by ID

PUT /api/movies/1
{
  "title": "The Shawshank Redemption",
  "year": 1994,
  "genre": "Drama"
}

Delete a movie by ID

DELETE /api/movies/1
